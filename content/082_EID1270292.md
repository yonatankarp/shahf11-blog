# אני אוהבת (לאו דווקא בסדר הזה)

1 ) ריח של דשא קצוץ אחרי הכיסוח.
2 ) ריחות של בישולים בכניסה לבית.
3 )אנשים שלא לוקחים את עצמם ברצינות יתר.
4 )חברים שיודעים לא רק לתמוך בעת צרה. אלא לפרגן כשטוב ומקסים (כן,כן)
5 )צבעים ושילובי צבעים.
6 )את הסתיו,את החורף, את האביב. ואת מה שעכשיו...
7 )את הרגע הזה שבו ילד מתחיל לדבר, ואחר כך לקרא.
8 ) לשבת על המרפסת. ולהרגיש שאני על "גג העולם".
9 )להסתכל בחנויות ראווה מיוחדות.
10 ) להכנס לחנויות מיוחדות.
11 ) לקנות בחנויות מיוחדות.
12 ) לבשל כשנחה עלי הרוח.ולהרגיש כמו "אם השנה"
13 ) לחלום (והרבה)
14 ) לנסוע רחוק ולשמוע ברקע שירים שאני אוהבת.
15 ) לפרום את הצמות של הקטנה ולהסניף את ריח השמפו.
16 ) כשנעזרים בי. כשמתייעצים איתי.
17 ) שאנשים אהובים עלי מחבקים אותי. (ממש צורך קיומי שלי).
18 ) תיקים. בכל הצורות ובכל הגדלים.
19 ) לשנות את הבית מידי כמה חודשים.
20 ) את ריח החג בבית של אמא ואבא.
21 ) את הים התיכון. וביחוד את חוף הבונים.
22 )סרטים טובים
23 )ספרים טובים
24 ) להאמין שמישהו למעלה דואג לי. ושהכל יהיה בסדר.
25 ) לישון. ואף יודעת לעשות זאת היטב. בכל תנאי ובכל מקום.
26 )לטוס
27 ) לנסוע לתל-אביב ולחזור בערב לחלקת האלוהים שלי.
28 ) אנשים עם חוש הומור. (לא מספרי בדיחות).
29 ) לשנות תספורות וצבעים בתדירות גבוהה.
30 ) שיחות נפש עם אהובי.
31 ) להתבונן מן הצד. בכלל... אוהבת להיות יותר בצד מאשר במרכז.
32 )את השעות הראשונות אחרי שהעוזרת עוזבת. והכל נראה ומריח נפלא.
33 ) מפגשים עם משפחה. עם אהובים.
34 )להקשיב. ופחות לדבר.
35 )לקום מוקדם (כשאני מצליחה)
36 ) עיסויים.
37 ) לשמוע צחוק בבית.
38 ) להרדים את הקטנה.
39 ) מקלחת חמה חמה בחורף. (בכלל לא אוהבת כשקר לי)
40 )שתיה עם קרח.
41 ) שוקולד ועוגות קרם בכל צורה.
42 )לחזור הביתה בסופו של יום.

בטח יש עוד... אבל החלטתי לכתוב על אהבות כמניין שנותי.
יום נפלא לכולם!!!
